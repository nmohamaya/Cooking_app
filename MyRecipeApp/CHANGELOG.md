# Changelog

All notable changes to MyRecipeApp will be documented in this file.

## [1.0.0] - 2025-12-19

### Added
- AI-powered recipe extraction from text using Claude API
- Multi-timer widget for tracking multiple cooking times simultaneously
- User feedback modal for recipe extraction improvements
- Recipe management (save, edit, delete recipes)
- Async storage for persistent recipe data
- Comprehensive test suite with 82.75% code coverage

### Fixed
- Jest test infrastructure for React Native environment
- Native module mocking for seamless testing

### Released
- Initial production release
- Android APK for Google Play Store
- Optimized performance and stability
