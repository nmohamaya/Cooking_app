# MyRecipeApp - Google Play Store Listing

This directory contains all the text content, descriptions, and metadata for the Google Play Store app listing.

## Files

- **app_title.txt** - App name for Play Store (max 50 characters)
- **short_description.txt** - Short description (max 80 characters)
- **full_description.txt** - Complete app description (max 4000 characters)
- **content_rating_questionnaire.md** - Content rating information
- **store_listing_guide.md** - Complete guide for entering data in Play Console

## Quick Reference

### App Title
Maximum 50 characters. Should be clear and searchable.

### Short Description
Maximum 80 characters. Appears under app name in search results.

### Full Description
Maximum 4000 characters. Use this to:
- Highlight key features
- Explain how to use the app
- Mention privacy and security
- Include call-to-action

### Category
Select: Food & Drink or Lifestyle

### Content Rating
Complete the questionnaire based on app content.

## Entering Data in Play Console

1. Go to **App content** section
2. Add all text from this directory
3. Upload screenshots and graphics
4. Add privacy policy URL
5. Review and publish

See **store_listing_guide.md** for detailed instructions.
